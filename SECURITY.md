# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in clix, please report it responsibly.

**Email:** dhicham.pro@gmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I'll respond within 48 hours.

## Scope

clix uses cookie-based authentication. Never share your auth tokens or cookies. Report any issue that could lead to credential exposure.
